## Instruções de intalação e execução

### Pré-requisitos

- Python 3.11
- Poetry

### Instalação

```bash
poetry install
```

### Execução

```bash
poetry run python main.py --prototxt mobilenet_ssd/MobileNetSSD_deploy.prototxt --model mobilenet_ssd/MobileNetSSD_deploy.caffemodel --input videos/example_01.mp4
```
